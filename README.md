# Kanban Board
Projeto criado para aprendizagem do framework React JS. 

## Funcionalidades
- Edição da tarefa;
- Alteração do status da tarefa (backlog, to do, in progress, completed);
- Deletar a tarefa;
- Drag n' drop (a implementar).

## Tecnologias
- React JS
- CSS (para estilização dos componentes)
